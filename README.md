# Movie Recommendation Full Stack

An AI-powered movie recommendation system built using Machine Learning and Full Stack Development.

## Features

- AI movie recommendations
- Beautiful cinematic UI
- Flask REST API
- Responsive frontend
- Full stack deployment

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Vercel

### Backend
- Flask
- Scikit-learn
- Render

### Machine Learning
- Cosine Similarity
- CountVectorizer
